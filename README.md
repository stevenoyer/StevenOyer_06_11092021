# Hot Takes - OpenClassrooms
Hot Takes est une application web de critique des sauces piquantes appelée « Hot Takes ».

## Projet
Ce projet a pour but de créer une API Rest.
- JS
- NodeJS

Accès au projet [Hot Takes](https://piiquante.stevenoyer.fr/)
